<h1 style="text-align: center">AUTUMN 管理系统-vue前端</h1>

#### 项目简介
一个基于 JDK1.8、MySQL8.0、Spring Boot 2.5、Mybatis Plus、JWT、Spring Security、Redis、Vue2.6、vuex、element-ui、sockJS的前后端分离的前台管理系统

#### 使用
第一次运行时，在终端执行（下载依赖）
```
npm install
```

启动项目
```
npm run serve
```

打包项目
```
npm run build
```
